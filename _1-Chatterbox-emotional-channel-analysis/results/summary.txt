METHODOLOGY DIFFERENCE:
Zonos uses 8D emotion vector with per-emotion conditioning weights.
Chatterbox uses scalar exaggeration (0.0–1.0).
This means Chatterbox cannot isolate individual emotions — it only controls intensity.
The exaggeration parameter affects all acoustic dimensions simultaneously, which may explain
why pause-related features dominate both the heatmap and classifier: higher exaggeration
values produce more dramatic prosodic breaks, while lower values produce flatter, more
monotone delivery with fewer pauses. This is a fundamentally different control mechanism
than Zonos's per-emotion vector conditioning.

NOTE ON NORMALIZATION:
Original z-score normalization (divide by |neutral| + 1e-8) produced Pauses z-scores
in the millions because neutral baseline pause features are near-zero. Fixed by using
a denominator floor at 10% of global feature std. Post-fix Pauses z-scores range
2.9–7.5, which is high but interpretable. The classifier result (0.457) is unaffected
by normalization — it operates on the same normalized features regardless of scale.

INTERPRETATION:
Chatterbox's scalar exaggeration collapses onto timing/pauses as the primary expressive
lever. A scalar control cannot route expressiveness to specific channels the way an 8D
vector can, so it defaults to the bluntest available signal (pauses and tempo).
Zonos routes to timbre. That is the architectural comparison worth highlighting.
