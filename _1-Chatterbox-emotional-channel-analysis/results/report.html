This report applies the same acoustic channel analysis method to Chatterbox and compares the results to the original Zonos findings.

Methodology

1. Audio Generation

5 neutral sentences were spoken by Chatterbox in 7 emotion conditions. Chatterbox has one control parameter called "exaggeration" (range 0.0 to 1.0). Each emotion was mapped to a value:

neutral=0.3, happy=0.7, sad=0.4, angry=0.9, fearful=0.8, surprised=0.85, low_energy=0.1

Random seed was fixed at 42 for all generations. Total: 35 WAV files.

2. Feature Extraction

17 acoustic features were extracted per WAV file using librosa, grouped into 6 channels:

ChannelFeatures
Pitchf0_mean, f0_std, f0_range, voiced_ratio
Tempoduration
Energyrms_mean, rms_std
Pausespause_count, pause_total, pause_max
Timbrespectral_centroid, mfcc_mean, mfcc_std
Voice qualityjitter_proxy, shimmer_proxy

3. Normalization

Neutral-relative z-scores were computed per sentence. A denominator floor (10% of global standard deviation) prevents division-by-zero when the neutral baseline is near zero. Constant features (words_per_sec, syllables_per_sec) were set to z=0.

4. Ablation Classifier

A RandomForest (100 trees) was trained on each channel group separately. Validation used LeaveOneGroupOut, where groups are sentence IDs. This tests whether each channel alone can predict the emotion condition.

5. Key Difference from Zonos

Zonos uses an 8D emotion vector with independent weight per emotion. Chatterbox uses a single scalar exaggeration value. Chatterbox cannot isolate individual emotions -- it only controls overall expressiveness intensity.

Key Findings

Finding 1: The mismatch does not replicate

The original study found that in Zonos, timbre moves most physically but pitch classifies best. In Chatterbox, pauses both move most and classify best (0.457). The mismatch between physical movement and classification does not appear here.

Finding 2: Scalar control collapses onto timing

Chatterbox has one control knob (exaggeration). It cannot route expressiveness to specific acoustic channels. The model defaults to timing and pauses as its primary expressive signal. Higher exaggeration creates more dramatic prosodic breaks. Lower exaggeration creates flatter, more monotone delivery.

Finding 3: All channels beat random baseline

Every channel scored above 0.143 (random). This means Chatterbox's exaggeration parameter does affect all acoustic dimensions, just to different degrees. Pauses (0.457) and Tempo (0.400) are the strongest signals. Pitch (0.314) and Timbre (0.314) are moderate. Energy (0.286) is the weakest.

Finding 4: Architecture shapes the emotion-acoustic relationship

Zonos (8D vector) routes emotion to timbre. Chatterbox (scalar) routes emotion to pauses. This is not a quality judgment. It shows that model design determines which acoustic channels carry emotion information.
